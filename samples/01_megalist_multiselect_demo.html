Megalist Multiselect Demo

This list has 100k items as generated json array:

First create the HTML Structure, then instantiate the list via javascript, and add an event handler:

Markup

          <div id="my_megalist" ></div>

Script

        $(document).ready( function() {
            $('#my_megalist').attr('data-provider-src', "[ "+createDataProvider(0).join(',')+" ]");
            $('#my_megalist').attr('data-provider-dst', "[ "+createDataProvider(1).join(',')+" ]");
            $('#my_megalist').megalist();
        });
        function createDataProvider(offset) {
            var result = [],
                quantity = 5000;
            for (x = quantity * offset; x < quantity * (offset + 1); x++) {
                result.push(
                    '{ "label": "item ' + x + '", "listValue":' + x +'}'
                );
            }
            return (result);
        }
